# mini_bash
A re-creation of bash using execve for school

<bold>How to run mbash :</bold>

1-download readline library : <code>sudo apt-get install libreadline-dev</code>

2-compile
<code>gcc -o mbash ./mbash.c -lreadline</code>

3-run
<code>./mbash</code>
